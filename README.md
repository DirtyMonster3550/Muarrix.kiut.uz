# KIUT Ilmiy Nashrlar

Платформа для управления научными публикациями Kimyo International University in Tashkent.

## Возможности
- Регистрация и авторизация авторов
- Подача статей в журналы KIUT (STEM, FINECS, Conferences)
- Панель администратора для рецензирования статей
- Email-уведомления при одобрении/отклонении статьи
- Уведомления в личном кабинете

## Быстрый старт

```bash
# Установить зависимости
npm install

# Запустить сервер
npm start
```

Сайт откроется на: **http://localhost:3000**

## Настройка Email (для отправки писем авторам)

Откройте файл `.env` и заполните данные вашей почты:

```env
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password   # Пароль приложения Gmail (не основной пароль!)
EMAIL_FROM=KIUT Ilmiy Nashrlar <your_email@gmail.com>
```

### Как получить пароль приложения Gmail:
1. Войдите в аккаунт Google
2. Перейдите в "Безопасность" → "Двухэтапная аутентификация" (включите)
3. Создайте "Пароль приложения" для "Почта"
4. Скопируйте 16-значный код в `EMAIL_PASS`

## Структура проекта

```
├── server.js           # Основной сервер
├── .env                # Конфигурация (email, пароли)
├── db/
│   └── database.js     # SQLite база данных
├── routes/
│   ├── auth.js         # Регистрация, вход
│   ├── submissions.js  # Подача статей
│   └── admin.js        # Панель администратора
├── utils/
│   └── mailer.js       # Отправка email
├── public/
│   ├── index.html      # Главная страница
│   ├── register.html   # Регистрация
│   ├── login.html      # Вход
│   ├── dashboard.html  # Кабинет автора
│   └── admin.html      # Панель администратора
└── uploads/            # Загруженные файлы статей
```
